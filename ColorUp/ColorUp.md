色相

范围: -1.0 ~ 1.0
默认值: 0.0
作用: 旋转整个色彩环
详细说明:
正值 → 色彩向红色→黄色→绿色方向偏移
负值 → 色彩向紫色→蓝色→青色方向偏移
1.0 = 360°旋转（回到原色）
常用于：调整整体氛围（暖调/冷调）
饱和度

范围: 0.0 ~ 2.0
默认值: 1.0
作用: 控制色彩鲜艳程度
详细说明:
0.0 → 完全灰度（黑白）
0.5 → 色彩减半
1.0 → 原始饱和度
1.5 → 饱和度增加50%
2.0 → 色彩非常鲜艳（可能超出显示范围）
常用于：增强画面活力或创建怀旧效果
亮度

范围: 0.0 ~ 2.0
默认值: 1.0
作用: 调整整体明暗程度
详细说明:
0.0 → 完全黑色
0.5 → 亮度减半
1.0 → 原始亮度
1.5 → 亮度增加50%
2.0 → 非常明亮（可能过曝）
常用于：调整画面整体明暗
对比度

范围: 0.0 ~ 2.0
默认值: 1.0
作用: 调整明暗区域的差异程度
详细说明:
0.0 → 完全灰色（无对比度）
0.5 → 对比度减半
1.0 → 原始对比度
1.5 → 对比度增加50%（亮部更亮，暗部更暗）
2.0 → 高对比度（可能丢失细节）
常用于：增强画面立体感

LUT强度

范围: 0.0 ~ 1.0
默认值: 1.0
作用: 控制LUT查找表的影响程度
详细说明:
0.0 → 完全使用原始颜色
0.5 → LUT和原始颜色各占50%
1.0 → 完全应用LUT效果
LUT（Look Up Table）是预设的色彩映射表
常用于：快速应用电影感、艺术风格

曝光值

范围: 0.0 ~ 5.0
默认值: 1.0
作用: 控制画面整体曝光程度
详细说明:
0.0 → 完全黑暗
0.5 → 曝光减半（变暗）
1.0 → 标准曝光
2.0 → 曝光加倍（变亮）
5.0 → 最大曝光（可能严重过曝）
常用于：模拟不同光照条件
伽马矫正系数

范围: 0.1 ~ 3.0
默认值: 2.2
作用: 调整色彩空间的非线性响应
详细说明:
< 2.2 → 暗部细节更明显，整体偏亮
= 2.2 → 标准Gamma（sRGB标准）
2.2 → 暗部更暗，对比度增强
常用于：校正显示设备差异或创造特殊效果
