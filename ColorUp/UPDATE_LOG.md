# ColorUp 更新日志

## 版本 1.1.0 - 2026-02-07

### 🎨 UI/UX 重大改进

#### 1. 底部导航栏设计
- ✅ **改变**: 将"预设风格/手动调整"从内容区域改为原生底部 TabView
- ✅ **优势**: 
  - 符合 iOS 原生交互习惯
  - 更容易单手操作
  - 图片显示区域更大
  - 标签切换更流畅

#### 2. 预设风格优化
- ✅ **网格布局**: 从横向滚动改为 3 列垂直滚动网格
  - 更容易浏览所有预设
  - 一次可以看到更多选项
  - 支持上下滑动查看
  
- ✅ **预览图片缩小**: 
  - 从 140x100 缩小到响应式比例（4:3）
  - 卡片 padding 从 12 减至 8
  - 字体大小优化（中文 11px，英文 8px）
  - 更紧凑的布局，屏幕可显示更多内容

- ✅ **分类标签优化**:
  - 字体从 14px 减至 12px
  - Padding 从 16/8 减至 14/6
  - 更节省空间的设计

#### 3. 手动调整面板改进
- ✅ **垂直滚动**: 改为 ScrollView，支持上下滑动
  - 所有参数一次展示
  - 更直观的浏览方式
  - 适配不同屏幕尺寸
  
- ✅ **布局优化**:
  - 增加间距到 20/24px
  - 标题字体从 24px 减至与其他视图一致
  - ACES 开关使用 GlassMorphicCard 包裹，视觉更统一

#### 4. 图片显示区域优化
- ✅ **更大的显示空间**: 
  - 图片区域固定在顶部
  - 移除了 GeometryReader，使用更简洁的布局
  - 自动适应可用空间

- ✅ **对比按钮优化**:
  - 尺寸缩小（字体 12px）
  - Padding 减小（12/6）
  - 更不显眼但仍然易用

- ✅ **空状态优化**:
  - 图标从 80px 减至 60px
  - 字体从 20px 减至 16px
  - 视觉更平衡

- ✅ **加载状态改进**:
  - 添加"处理中..."文字提示
  - 更友好的用户反馈

### 📱 交互改进总结

#### 之前的布局：
```
┌─────────────────────────┐
│      导航栏              │
├─────────────────────────┤
│                         │
│    图片显示区域          │
│    (固定 400px)         │
│                         │
├─────────────────────────┤
│ [预设] [手动] 标签切换   │
├─────────────────────────┤
│                         │
│   内容区域              │
│   (横向滚动预设)        │
│                         │
└─────────────────────────┘
```

#### 现在的布局：
```
┌─────────────────────────┐
│      导航栏              │
├─────────────────────────┤
│                         │
│   图片显示区域           │
│   (自适应空间，更大)     │
│                         │
├─────────────────────────┤
│                         │
│   内容区域              │
│   (垂直滚动网格/列表)   │
│                         │
├─────────────────────────┤
│ [预设] 👁️  [手动] 🎚️    │  ← 底部 TabView
└─────────────────────────┘
```
