# ColorUp - 色彩魔法工坊

一个专业的图片色彩调整应用，采用原生 SwiftUI 构建，提供 20+ 预设风格和全面的手动调整功能。

## ✨ 主要功能

### 🎨 20+ 专业预设风格

- **复古科技**：早期黑白电视、早期彩色电视、NTSC制式
- **艺术创作**：水墨画、铅笔画、版画木刻
- **胶片模拟**：复古胶片70s、柯达胶卷、富士胶卷
- **特殊效果**：红外线摄影、X光片、夜视仪
- **现代风格**：Instagram褪色、赛博朋克、高级灰、莫兰迪色系
- **游戏风格**：生存恐怖、末日废土、历史模式
- **工业设计**：PANTONE年度色

### 🔧 精细手动调整

- **色相调整** (-1.0 ~ 1.0)：旋转整个色彩环，调整冷暖色调
- **饱和度** (0.0 ~ 2.0)：控制色彩鲜艳程度，从灰度到超饱和
- **亮度** (0.0 ~ 2.0)：调整整体明暗程度
- **对比度** (0.0 ~ 2.0)：增强或减弱明暗区域差异
- **曝光值** (0.0 ~ 5.0)：模拟不同光照条件
- **伽马矫正** (0.1 ~ 3.0)：调整色彩空间非线性响应
- **ACES色调映射**：专业电影级色彩处理

### 🎭 实时对比预览

- 长按查看原图与处理后图片对比
- 流畅的动画过渡效果
- 实时参数反馈

### 💾 导出功能

- 保存处理后的图片到相册
- 保持原始图片质量

## 🚀 快速开始

### 前置要求

- Xcode 15.0+
- iOS 17.0+
- Swift 5.9+

### 权限配置

在 `Info.plist` 中添加以下权限：

```xml
<key>NSPhotoLibraryUsageDescription</key>
<string>需要访问您的相册以选择和保存图片</string>
<key>NSPhotoLibraryAddUsageDescription</key>
<string>需要权限以保存处理后的图片到相册</string>
```

### 运行项目

1. 打开 `ColorUp.xcodeproj`
2. 选择目标设备（iOS 模拟器或真机）
3. 点击运行按钮 (⌘R)

## 📖 使用指南

### 基础操作

1. **选择图片**：点击导航栏的"选择"按钮，从相册中选择一张图片
2. **应用预设**：在"预设风格"标签页中选择任意风格，即时预览效果
3. **手动调整**：切换到"手动调整"标签页，精细调整各项参数
4. **对比预览**：点击图片右下角的眼睛按钮查看原图
5. **保存图片**：满意后点击"保存"按钮导出到相册
6. **重置参数**：点击"重置"按钮恢复默认设置

### 高级技巧

- **快速切换预设**：横向滑动预设卡片快速浏览
- **分类筛选**：使用顶部的分类标签快速定位风格
- **精细调整**：拖动滑块时会实时显示数值变化
- **组合使用**：先应用预设，再微调参数，创造独特风格

## 🎨 色彩参数详解

### 色相 (Hue)
- **范围**：-1.0 ~ 1.0
- **作用**：旋转色彩环，改变整体色调
- **效果**：正值偏暖，负值偏冷

### 饱和度 (Saturation)
- **范围**：0.0 ~ 2.0
- **作用**：控制色彩鲜艳度
- **效果**：0.0 为灰度，2.0 为超饱和

### 亮度 (Brightness)
- **范围**：0.0 ~ 2.0
- **作用**：调整整体明暗
- **效果**：0.0 为全黑，2.0 为过曝

### 对比度 (Contrast)
- **范围**：0.0 ~ 2.0
- **作用**：调整明暗差异
- **效果**：增强画面立体感

### 曝光值 (Exposure)
- **范围**：0.0 ~ 5.0
- **作用**：模拟相机曝光
- **效果**：影响画面整体亮度分布

### 伽马矫正 (Gamma)
- **范围**：0.1 ~ 3.0
- **作用**：调整色彩空间响应曲线
- **效果**：改变暗部细节表现

### ACES 色调映射
- **开关**：启用/禁用
- **作用**：电影级色彩处理
- **效果**：更自然的色彩过渡
