## 🔧 技术实现

### 1. 刻度标签动态定位

**之前的实现**：
```swift
VStack(spacing: 0) {
    ForEach(ticks.reversed(), id: \.self) { tick in
        Text(formatTickLabel(tick))
            .frame(height: 20)  // 固定高度
    }
}
```

**问题**：
- 刻度间隔固定为 20px
- 与滑条高度不匹配
- 位置不精确

**现在的实现**：
```swift
GeometryReader { geometry in
    ZStack(alignment: .bottom) {
        ForEach(ticks.reversed(), id: \.self) { tick in
            Text(formatTickLabel(tick))
                .offset(y: -tickPosition(for: tick, in: geometry.size.height))
        }
    }
}
```

**优势**：
- ✅ 动态计算每个刻度的精确位置
- ✅ 自动适应滑条高度
- ✅ 完美对齐刻度线

### 2. 刻度位置计算

```swift
private func tickPosition(for tick: Double, in totalHeight: CGFloat) -> CGFloat {
    let normalizedValue = (tick - range.lowerBound) / (range.upperBound - range.lowerBound)
    return totalHeight * normalizedValue  // 从底部开始的高度
}
```

**计算流程**：
1. 将刻度值归一化到 0-1 范围
2. 乘以总高度得到像素位置
3. 使用负偏移从底部开始定位

**示例**（色相范围 -1.0 ~ 1.0，高度 200px）：
| 刻度值 | 归一化值 | 像素高度 | 显示位置 |
|--------|---------|---------|---------|
| -1.0   | 0.0     | 0px     | 底部 |
| -0.5   | 0.25    | 50px    | 底部+50px |
| 0.0    | 0.5     | 100px   | 中间 |
| 0.5    | 0.75    | 150px   | 顶部-50px |
| 1.0    | 1.0     | 200px   | 顶部 |

### 3. 智能点击检测

**之前的实现**：
```swift
.onTapGesture { location in
    // 点击任意位置都设置值
    let newValue = ...
    value = newValue
}
```

**问题**：
- 点击手柄会误触发
- 拖动手柄时可能触发点击

**现在的实现**：
```swift
.onTapGesture { location in
    // 计算手柄位置
    let handleY = geometry.size.height - progressHeight(in: geometry.size.height)
    let handleRadius: CGFloat = 12
    
    // 检查是否点击在手柄范围内
    if abs(location.y - handleY) > handleRadius {
        // 仅在点击轨道时设置值
        let newValue = ...
        value = newValue
    }
}
```

**优势**：
- ✅ 点击手柄不会设置值
- ✅ 仅点击轨道时响应
- ✅ 12px 半径容错范围
- ✅ 避免误操作

## 📐 布局逻辑

### 整体结构

```swift
VStack {
    // 上方：滑条和刻度
    HStack {
        GeometryReader { /* 刻度标签 */ }
        GeometryReader { /* 垂直滑条 */ }
    }
    .frame(height: 200)
    
    // 下方：标题和数值
    VStack {
        HStack { /* 图标 + 标题 */ }
        Text { /* 数值显示 */ }
    }
}
.frame(width: 80)
```

### 对齐方式

- **刻度标签**：`ZStack(alignment: .bottom)` 从底部对齐
- **滑条**：`ZStack(alignment: .bottom)` 从底部对齐
- **标题和数值**：`VStack(alignment: .center)` 居中对齐
