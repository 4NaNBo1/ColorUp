# ColorUp 快速开始指南 🚀

## ⚡ 立即开始（3 分钟）

### 第 1 步: 配置权限（必需！）

在 Xcode 中打开项目后，需要添加相册访问权限：

1. 点击项目导航器中的 `ColorUp` 项目
2. 选择 `ColorUp` target
3. 切换到 **Info** 标签页
4. 点击任意行的 `+` 按钮，添加以下两项：

| 键 | 值 |
|---|---|
| `Privacy - Photo Library Usage Description` | `需要访问您的相册以选择图片` |
| `Privacy - Photo Library Additions Usage Description` | `需要权限以保存处理后的图片到相册` |

### 第 2 步: 运行项目

1. 选择目标设备（建议使用 iOS 17+ 模拟器或真机）
2. 按 `⌘R` 运行
3. 首次运行时授予照片访问权限

### 第 3 步: 开始使用

1. 点击"选择"按钮，从相册选择一张图片
2. 在"预设风格"中选择任意风格
3. 或切换到"手动调整"精细调节
4. 点击"保存"导出到相册

## ✨ 核心功能

### 7 项手动调整参数

- **色相** (Hue): 调整冷暖色调
- **饱和度** (Saturation): 控制色彩鲜艳度
- **亮度** (Brightness): 整体明暗调节
- **对比度** (Contrast): 增强画面立体感
- **曝光值** (Exposure): 模拟相机曝光
- **伽马矫正** (Gamma): 调整暗部细节
- **ACES 色调映射**: 电影级色彩处理

## 🎯 使用技巧

### 快速上手
- 💡 先尝试预设风格，找到接近的效果
- 💡 再用手动调整进行微调
- 💡 点击眼睛图标对比原图

### 参数调整建议

#### 打造复古效果
```
降低饱和度 → 0.3 ~ 0.6
降低对比度 → 0.7 ~ 0.9
偏暖色调 → 色相 +0.05 ~ +0.1
```

#### 增强画面冲击力
```
提高饱和度 → 1.2 ~ 1.5
提高对比度 → 1.3 ~ 1.7
适当提亮 → 亮度 1.1 ~ 1.3
```

#### 营造艺术感
```
极低饱和度 → 0.1 ~ 0.3
高对比度 → 1.5 ~ 2.0
调整伽马 → 1.4 ~ 1.8
```

## ⚠️ 重要提醒

### 必须配置的项目
- ❗ Info.plist 相册权限（见第1步）
- ❗ Xcode 开发者签名

### 系统要求
- ✅ iOS 17.0 或更高版本
- ✅ Xcode 15.0 或更高版本
- ✅ Swift 5.9 或更高版本

### 建议
- 💡 使用真机测试获得最佳性能
- 💡 图片大小建议 < 10MB
- 💡 首次运行记得授予相册权限

## 🐛 遇到问题？

### 无法选择图片
→ 检查是否添加了 Info.plist 权限  
→ 在系统设置中确认已授权

### 保存失败
→ 检查相册添加权限  
→ 确认存储空间充足

### 处理缓慢
→ 尝试使用较小的图片  
→ 在真机上运行效果更好

### 编译错误
→ 清理构建文件夹 (⇧⌘K)  
→ 检查 Xcode 版本 >= 15.0
